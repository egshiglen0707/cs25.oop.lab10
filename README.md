Заавар:

1. Энэ репог fork хийнэ үү.
2. Дасгал тус бүрт кодыг хэрэгжүүлнэ.
3. Commit хийж, надад линкээ явуулна уу.

## Хийсэн зүйлс

- `Zoo.java` файлд амьтдыг нэг жагсаалтаас нөгөө жагсаалт руу шилжүүлэх метод бичсэн.
- `Library.java` файлд дотоод `BookShelf` класс дээр iterator хэрэгжүүлсэн.
- `StringProcessor.java` файлд `Predicate`, `Function`, `Consumer` ашиглаж үг боловсруулах код бичсэн.
- `ProductAnalyzer.java` файлд stream ашиглаж бүтээгдэхүүнийг шүүж, эрэмбэлж, төрлөөр нь бүлэглэсэн.
